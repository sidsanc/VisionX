# VisionX
Install python 3.8, pip3

Checkout repo: 
https://github.com/sidsanc/VisionX


Set Up venv

python3 -m venv ./
or 
python -m venv ./


Bash/MacOS
source ./bin/activate

for windows
venv\Scripts\activate.bat


pip3 install -r requirements.txt

To run application locally:
python3  manage.py run
